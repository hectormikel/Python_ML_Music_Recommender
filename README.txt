# ML Music Recommender Project

**Version 1.0.0**

This project predicts and recommends music genre to the user depending on the user profile. CSV data is imported and contains data on the person's age, gender, and recommended music genre. A model is created and trained to recommend music genre to the user based on age and gender. Decision Tree Classifier algorithm is used. The data is split for training and testing, with test size = 0.2 (20%), or train size = 0.8. Accuracy score of the model is shown as variable score. The decision tree graphic is exported and can be presented via Visual Code, Graphviz (dot) extension.
